
### flow & type script

### 01.成本

    flow 20至30分鐘。

    type script 2小時至3小時

### 02.功能

   type script 延伸性強，大於 flow
   如：react , anglar, 明顯 github 的資源多些

   flow 只是單純的內文檢查器。

   typescript 是 整個代碼再轉成 js。
