## 百度糯米前端技术学院

百度糯米前端技术学院，是覆盖糯米业务范围最广的前端技术团队，包含B(Business)&C(Customs)两端，应用场景涵盖PC、NA和WAP三端。团队内有对前端各个领域擅长的资深技术人才，从入门到精通都有专门导师做1对1指导。学院的宗旨是服务学员，专注如一。欢迎大家加入百度糯米前端技术学院，跟我们一起，探索前端技术奥妙！
